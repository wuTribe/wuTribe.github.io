面试题 02.07. 链表相交

代码

两种方法：

  1. 把 A 链表地址存下，再遍历 B 链表逐个比较，相同就返回。代码简单就不写了
  2. 双指针，双遍历。这里需要分类讨论
    • 如果相交
      • 假设两个链表分别为 a,b
        a.length = a1 + a2
        b.length = b1 + b2
        当 a b 在 c 处相交时，a2 == b2，即
        a.length = a1 + c
        b.length = b1 + c
        当 a 遍历完走 b，b 遍历完走 a，他们的长度为
        a：a1 + c + b1 + c
        b：b1 + c + a1 + c
        如果 a1 == b1，第一个 c 就相遇
        如果 a1 != b1，第二个 c 相遇
        所以，当他们走完对方的独立长度时，就是二者相交点
    • 如果不相交
      • 遍历两次都找不到公共点

这个题目给我的感觉，优化方案更像是数学题，能想到就很简单，想不到就死活做不出来
